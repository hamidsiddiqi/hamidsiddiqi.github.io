Owl Illustration

Using my knowledge of Adobe Illustrator, I designed an illustration of an owl sitting in a tree. I applied many skills in this project such as using the pen tool to create shapes for the owl and the tree, adding a gradient to the tree, using the shape builder tool to merge different shapes on the owl’s body, and using the blend tool to make copies of the tree textured lines.
